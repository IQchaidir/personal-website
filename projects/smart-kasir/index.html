SMART KASIR

Repo Github

SmartKasir is an application designed for conducting product purchase transactions. It has two roles for managing the application: Cashier and Admin. In this context, Cashiers create transactions by adding products to be purchased and then selecting which products to buy. Meanwhile, Admins manage products and Cashiers.
